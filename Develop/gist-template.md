# Regex Gist - Matching a URL

This is an overview of a regular expression for matching a url.

## Summary
I will explain each piece of a regular expression for matching a URL. The regex is as follows:
/^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$/
